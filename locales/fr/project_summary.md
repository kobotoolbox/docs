# Récapitulatif du projet

L'onglet **Récapitulatif du projet** vous offre un affichage visuel de toutes les informations générales relatives à votre projet. Selon que vous êtes le propriétaire du projet ou un utilisateur disposant de permissions restreintes, différentes informations vous seront présentées. Le récapitulatif du projet comprend :

   1. **Description** : Inclut toutes les métadonnées du projet.
   2. **Soumissions** : Un décompte et un graphique des soumissions (sur les 31 derniers jours).
   3. **Détails du formulaire** : Inclut la date de dernière modification, la dernière soumission et le nombre de questions.
   4. **Liens rapides** : Liens utiles liés à la collecte de données pour le projet.
   5. **Données** : Liens utiles vers les informations relatives aux données soumises.
   6. **Membres de l'équipe** : Liste des utilisateurs ayant accès au projet.
