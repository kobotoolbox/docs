# Réinitialiser votre mot de passe
**Dernière mise à jour :** <a href="https://github.com/kobotoolbox/docs/blob/855616c6ac4634f2dc835438697aab6869c7f11a/source/reset_password.md" class="reference">22 Sep 2023</a>


Si vous avez oublié ou perdu le mot de passe de votre compte, KoboToolbox vous permet de le réinitialiser facilement.

Pour réinitialiser votre mot de passe :

-   Pour les organisations utilisant le **serveur KoboToolbox mondial** :
    Rendez-vous sur
    [kf.kobotoolbox.org](https://kf.kobotoolbox.org)
-   Pour les organisations utilisant le **serveur KoboToolbox Union européenne** :
    Rendez-vous sur [eu.kobotoolbox.org](https://eu.kobotoolbox.org)

![Formulaire de connexion KoboToolbox](/images/resetting_passwords/login_form.png)

-   Cliquez sur **Mot de passe oublié ?** dans le formulaire de connexion.
-   Saisissez votre adresse email.
-   Cliquez sur **Réinitialiser le mot de passe**.

KoboToolbox vous enverra alors un email de réinitialisation de mot de passe suite à votre demande. Si vous n'avez pas reçu d'email contenant le lien de réinitialisation, essayez les solutions suivantes :

-   Vérifiez si l'email de réinitialisation de mot de passe se trouve dans votre dossier de courrier indésirable ou spam, car il est possible que votre fournisseur de messagerie l'y ait placé.
-   Vérifiez si vous avez créé votre compte avec une adresse email différente — cliquez à nouveau sur le lien « Mot de passe oublié ? » et saisissez votre autre adresse email.
-   Il est également possible que vous tentiez de réinitialiser votre mot de passe sur le mauvais serveur. Vérifiez l'URL dans votre barre d'adresse pour vous assurer que vous êtes bien sur le serveur sur lequel vous avez créé votre compte, que ce soit le [serveur KoboToolbox mondial](https://kf.kobotoolbox.org) ou le [serveur KoboToolbox Union européenne](https://eu.kobotoolbox.org).

Si vous n'avez toujours pas reçu l'email contenant le lien de réinitialisation après avoir essayé toutes les solutions proposées ci-dessus, veuillez envoyer un email avec pour objet « Password reset issues » à [support@kobotoolbox.org](mailto:support@kobotoolbox.org) en indiquant votre nom d'utilisateur, l'adresse email utilisée lors de votre inscription, et le serveur sur lequel vous avez créé votre compte.
