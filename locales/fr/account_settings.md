Les paramètres de votre compte KoboToolbox vous permettent de gérer vos informations de profil, d'accéder aux fonctionnalités de sécurité et de surveiller votre utilisation. Vous pouvez également passer à une offre supérieure ou acheter des modules supplémentaires pour bénéficier d'un espace de stockage additionnel et de fonctionnalités de transcription ou de traduction automatiques.

Pour accéder aux paramètres de votre compte :
1. [Connectez-vous](https://www.kobotoolbox.org/sign-up/) à votre compte KoboToolbox.
2. Cliquez sur l'icône de votre profil en haut à droite.
3. Cliquez sur **Paramètres du compte**.

![Gérer les paramètres du compte](images/account_settings/account_settings.png)

## Profil

La page <i class="k-icon-user"></i> **Profil** est la vue par défaut lorsque vous ouvrez les paramètres de votre compte.

Sur la page **Profil**, vous pouvez modifier les informations suivantes :
- Nom complet
- Pays et ville
- Secteur et type d'organisation
- Nom et site web de l'organisation
- Courte biographie et profil LinkedIn

Vous pouvez également vous inscrire à la newsletter KoboToolbox pour recevoir des mises à jour occasionnelles sur KoboToolbox.

### Supprimer votre compte
Vous pouvez également **supprimer votre compte KoboToolbox** depuis la page **Profil**. Pour supprimer votre compte :
1. [Supprimez](https://support.kobotoolbox.org/fr/managing_projects.html#deleting-projects) ou [transférez la propriété](https://support.kobotoolbox.org/fr/project_sharing_settings.html#transferring-ownership-of-a-project) de tous les projets appartenant à votre compte.
2. Dans **Paramètres du compte > Profil**, cliquez sur **Supprimer le compte**. Ce bouton sera désactivé si vous n'avez pas supprimé tous les projets appartenant à votre compte.
3. Saisissez votre nom d'utilisateur pour confirmer la suppression de votre compte. Cette action est irréversible une fois effectuée.

<p class="note">
  <strong>Note :</strong> Vous n'avez pas besoin de supprimer les projets partagés avec vous par d'autres utilisateurs avant de supprimer votre compte.
</p>

## Sécurité

Pour accéder aux fonctionnalités de sécurité, sélectionnez la page <i class="k-icon-lock-alt"></i> **Sécurité** dans le menu des paramètres du compte.

Sur la page **Sécurité**, vous pouvez :
- Modifier votre mot de passe.
    - Si vous connaissez votre mot de passe actuel, cliquez sur **Mettre à jour**.
    - Si vous avez oublié votre mot de passe actuel, cliquez sur **Vous avez oublié votre mot de passe ?** pour le [réinitialiser](https://support.kobotoolbox.org/fr/reset_password.html).
- Mettre à jour votre adresse email en cliquant sur **Modifier** à côté de **Adresse email**.
- Afficher ou régénérer votre **clé API**, requise pour certains types d'authentification avec l'[API KoboToolbox](https://support.kobotoolbox.org/fr/api.html).
- Activer l'[authentification à deux facteurs](https://support.kobotoolbox.org/fr/two_factor_authentication.html) (2FA) pour protéger votre compte.
- Consulter l'[activité récente du compte](https://support.kobotoolbox.org/fr/activity_logs.html#access-logs) et exporter les données de suivi.
- Vous déconnecter de tous les appareils connectés à votre compte.

## Utilisation

Pour surveiller l'utilisation de votre compte, sélectionnez la page <i class="k-icon-reports"></i> **Utilisation** dans le menu des paramètres du compte. Le suivi de l'utilisation du compte peut vous aider à respecter les [limites de votre plan](https://www.kobotoolbox.org/pricing/).

La page Utilisation comporte deux onglets :
- L'onglet **Account Total** affiche les limites de votre plan et l'utilisation totale du compte.
- L'onglet **Per Project Total** affiche l'utilisation pour chacun de vos projets.

Vous pouvez surveiller les types d'utilisation suivants :
| Type d'utilisation | Description |
| :--- | :--- |
| Soumissions | Le nombre total de soumissions collectées par les projets appartenant à votre compte au cours du [cycle de facturation](https://support.kobotoolbox.org/fr/glossary.html#billing-cycle) en cours. |
| Stockage de fichiers | La taille totale des photos, fichiers audio, vidéos et fichiers collectés avec vos formulaires ou importés dans **Paramètres > Médias** du projet. |
| Minutes de transcription | Le nombre de minutes de transcription utilisées par votre compte pour la [transcription automatique de fichiers audio](https://support.kobotoolbox.org/fr/transcription-translation.html). |
| Caractères de traduction | Le nombre de caractères de traduction utilisés par votre compte pour la [traduction automatique de transcriptions audio](https://support.kobotoolbox.org/fr/transcription-translation.html). |


## Plans

KoboToolbox propose différents plans adaptés à divers besoins en matière de collecte de données. Pour consulter un résumé des différents plans et passer à une offre supérieure, sélectionnez la page <i class="k-icon-editor"></i> **Plans** dans le menu des paramètres du compte.

Sur la page **Plans**, vous pouvez :
- Consulter votre plan actuel, notamment les quotas et l'utilisation des fonctionnalités.
- Passer au plan **Professional** ou **Teams**.
- Basculer entre les vues **Mensuel** et **Annuel** pour afficher les différents tarifs et limites des plans selon le cycle de facturation.

<p class="note">
  Pour en savoir plus sur les plans et les limites de KoboToolbox, consultez la page <a href="https://www.kobotoolbox.org/pricing/">Tarifs</a>.
</p>

## Modules supplémentaires

Les modules supplémentaires sont des packages que vous pouvez ajouter à votre plan actuel pour augmenter vos limites d'utilisation. Si vous approchez ou avez atteint les limites d'utilisation incluses dans votre plan, vous pouvez acheter des modules supplémentaires pour bénéficier d'une utilisation additionnelle.

<p class="note">
  <strong>Note :</strong> Les modules supplémentaires peuvent être ajoutés à votre plan à tout moment. Pour éviter toute interruption de la collecte de données, nous vous recommandons d'acheter des modules supplémentaires avant d'atteindre votre limite.
</p>

Pour acheter des modules supplémentaires pour votre compte, sélectionnez la page <i class="k-icon-plus"></i> **Modules supplémentaires** dans le menu des paramètres du compte.

Sur la page **Modules supplémentaires**, vous pouvez :
- Acheter un module supplémentaire **Stockage de fichiers** pour ajouter jusqu'à 50 Go d'espace de stockage à votre compte KoboToolbox, si vous utilisez le plan Community gratuit.
- Acheter un module supplémentaire **Pack NLP** pour augmenter vos minutes de [transcription](https://support.kobotoolbox.org/fr/transcription-translation.html) et vos caractères de [traduction](https://support.kobotoolbox.org/fr/transcription-translation.html).
- Acheter un module supplémentaire **Requêtes d'analyse automatique** pour augmenter votre quota d'[analyse qualitative](https://support.kobotoolbox.org/fr/qualitative_analysis.html) générée par IA.

<p class="note">
  <strong>Note :</strong> Si vous avez besoin d'augmenter vos limites de soumissions, <a href="https://www.kobotoolbox.org/contact">contactez</a> notre équipe d'assistance pour acheter un module supplémentaire de <strong>soumissions illimitées</strong>.
</p>

## Langue

Pour modifier la langue de votre interface KoboToolbox :
1. Accédez à la page d'accueil **Projets**.
2. Cliquez sur l'icône de votre profil dans le coin supérieur droit.
3. Cliquez sur <i class="k-icon-language"></i> **Langue** et sélectionnez votre langue préférée.
