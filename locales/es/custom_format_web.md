# Formato personalizado en formularios web

_Ten en cuenta que el formato personalizado no funciona en la aplicación Android KoboCollect._

Cuando usas formularios web (Enketo) como método de recolección de datos, tienes la opción de dar formato a las etiquetas de tus preguntas de varias maneras.

Para dar formato al texto de tus notas, usa una de estas opciones:

-   _enfatiza palabras_ envolviéndolas dentro de `_estas_` o `*estos caracteres*`
-   **enfatiza con mayor énfasis** las palabras envolviéndolas dentro de `__estos caracteres__` o `**estos**`
-   [un enlace](https://www.kobotoolbox.org) puede añadirse usando `[un enlace](https://www.kobotoolbox.org)`

Estos formatos no funcionarán en KoboCollect, que simplemente mostrará los caracteres `_` o `*` junto con el texto.

## Combinar texto y multimedia mediante notas consecutivas

Ya puedes incluir una imagen, video o archivo de audio en tus preguntas. (Esto funciona tanto en formularios web como en KoboCollect.) Pero cuando usas formularios web, puedes elegir mostrar la imagen junto al texto, por ejemplo, simplemente añadiendo dos notas consecutivas. Para obtener más información sobre este proceso, [consulta esta publicación](https://blog.enketo.org/better-notes).
