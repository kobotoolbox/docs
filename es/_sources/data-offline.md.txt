# Recolección de datos sin conexión

**Toda la recolección de datos puede realizarse sin conexión, tanto con KoboCollect como con formularios web.**

Cuando un usuario ingresa datos, estos se almacenan primero en el dispositivo. KoboCollect [puede configurarse](https://support.kobotoolbox.org/es/kobocollect_settings.html#form-management-settings) para intentar enviar la información a través de una conexión de red de inmediato o solo en una etapa posterior, cuando el encuestador o supervisor quiera cargar los formularios finalizados. Los formularios web siempre intentarán cargar los datos de inmediato y volverán a intentarlo hasta que se haya restablecido una conexión.

Toda la sincronización está protegida incluso contra una conexión a internet de baja calidad. Si una conexión se agota o se interrumpe mientras se transfiere un formulario específico, este se reenviará en el siguiente intento de carga. En este caso, el servidor no integrará datos recibidos de forma incompleta. Solo cuando un registro se haya cargado correctamente y el servidor confirme su recepción, los datos de la encuesta se eliminarán de la cola de carga.

Los formularios web utilizan el almacenamiento sin conexión HTML5 del navegador para guardar tanto las respuestas de la encuesta como el formulario en sí. Es importante esperar el pequeño mensaje de confirmación en la parte superior derecha del formulario, que mostrará una marca de verificación verde una vez que el formulario haya sido guardado en caché. Después de esto, se puede acceder al formulario e ingresar datos incluso sin ninguna conexión. Se recomienda crear un marcador en el dispositivo para acceder fácilmente a formularios específicos.

## ¿Qué hago si NO tengo acceso a internet?

Si no es posible conectar un dispositivo a internet (los encuestadores no pueden desplazarse a un área donde haya conexión disponible), también es posible transferir los datos de la encuesta desde KoboCollect mediante una herramienta externa, [ODK Briefcase](https://docs.getodk.org/briefcase-intro), conectando los dispositivos móviles por cable USB a un computador local. Para obtener más detalles sobre cómo usar ODK Briefcase para transferir datos, [lee esta publicación](https://blog.cartong.org/2016/03/11/migration-odk-platforms). Después de esto, es posible cargar los datos de la encuesta desde un computador centralizado al servidor de KoboToolbox, utilizando la misma herramienta.

Por último, también es posible instalar KoboToolbox en un computador local, por ejemplo, un portátil, y luego conectar los dispositivos móviles locales al computador a través de una red WiFi local. Esta red WiFi no necesita estar conectada a internet, ya que habría una conexión directa entre los dispositivos móviles y el computador local. [Visita este enlace](https://github.com/kobotoolbox/kobo-install) para saber cómo instalar KoboToolbox en tu computador.
