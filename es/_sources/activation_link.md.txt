# Solución de problemas con el link de activación de la cuenta
**Last updated:** <a href="https://github.com/kobotoolbox/docs/blob/0e5a784bc21dd79ac926f958b3cbb238d64ecc53/source/activation_link.md" class="reference">23 Sep 2025</a>


Después de crear tu cuenta, se enviará un link de activación a la dirección de correo electrónico que proporcionaste. Debes hacer clic en este link dentro de las 72 horas para activar tu cuenta.

Si no haces clic en el link dentro de tres días, o si el link ha expirado:
1. Intenta iniciar sesión con el nombre de usuario y la contraseña que creaste. Un nuevo link de activación debería enviarse automáticamente.
2. Si eso no funciona, el problema puede deberse a:
    * Un error tipográfico en la dirección de correo electrónico que ingresaste.
    * El dominio de correo electrónico de tu organización está bloqueando nuestros mensajes.
    * El correo de activación fue colocado en tu carpeta de spam o correo no deseado.
3. Si aún no puedes acceder a tu cuenta, contáctanos en [support@kobotoolbox.org](support@kobotoolbox.org).
