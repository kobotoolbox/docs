Introducción a KoboToolbox¶

Última actualización: 22 Jun 2026

KoboToolbox es una innovadora plataforma de código abierto para recolectar, gestionar y visualizar datos. Es la herramienta de recolección de datos más utilizada para iniciativas de impacto social en todo el mundo.

Para obtener más información sobre cómo KoboToolbox apoya a las organizaciones sin fines de lucro en la generación de impacto a través de los datos, ver Acerca de KoboToolbox: recolección de datos accesible.

Este artículo presenta KoboToolbox y destaca sus funcionalidades principales. Cubre las siguientes características clave:

  • Desarrollar formularios

  • Usar la biblioteca de preguntas

  • Recolectar datos

  • Gestionar y analizar envíos

  • Visualizar y descargar datos

  • Gestionar proyectos y equipos

  • Conectar KoboToolbox con otras plataformas

Para comenzar con la elaboración de formularios en KoboToolbox, ver nuestra Guía de inicio rápido.

Desarrollar formularios¶

El intuitivo editor de formularios de KoboToolbox (Formbuilder) permite a los usuarios crear formularios con diferentes tipos de preguntas, lógica de formulario avanzada y traducciones. Los formularios también se pueden crear y cargar usando XLSForm.

  • Tipos de preguntas: Elige entre más de 25 tipos de preguntas cualitativas y cuantitativas, incluyendo texto, selección múltiple, multimedia y GPS.

  • Lógica del formulario: Añade lógica de omisión, cálculos y criterios de validación para obtener datos de alta calidad.

  • Traducciones: Incluye fácilmente traducciones del formulario en cientos de idiomas diferentes.

  • XLSForm: Para crear formularios más complejos, usa XLSForm en tu editor de hojas de cálculo preferido, cárgalo en KoboToolbox y edítalo directamente en el Formbuilder.

Para comenzar a crear formularios en KoboToolbox, ver Comenzar con el editor de formularios.

Usar la biblioteca de preguntas¶

Estandariza los datos y simplifica la elaboración de formularios creando plantillas de formularios y usando preguntas de la biblioteca de Colecciones públicas.

  • Mi biblioteca: Crea, organiza y guarda preguntas y plantillas personalizadas en tu biblioteca.

  • Colecciones públicas: Usa una pregunta, un bloque de preguntas o una plantilla de formulario completa de la biblioteca de Colecciones públicas en tu formulario y personalízala según tus necesidades. Las preguntas y plantillas de la biblioteca pública son aportadas por otros usuarios y están disponibles para todos.

Para obtener más información sobre la biblioteca de preguntas de KoboToolbox, ver Usar la biblioteca de preguntas de KoboToolbox.

Recolectar datos¶

Recolecta datos sin conexión o en línea en cualquier dispositivo usando formularios web y la aplicación KoboCollect.

  • Formularios web: KoboToolbox usa formularios web para recolectar datos desde cualquier navegador de internet. Los formularios web se pueden usar en laptops, computadoras de escritorio o dispositivos móviles, con o sin conexión a internet.

  • KoboCollect: Usa la aplicación Android KoboCollect para recolectar datos en dispositivos móviles. Los formularios se pueden descargar en KoboCollect desde una cuenta de KoboToolbox para la recolección de datos sin conexión. Los datos recolectados en KoboCollect se pueden cargar luego en KoboToolbox. Descarga la aplicación KoboCollect de forma gratuita desde Google Play.

Para obtener más información sobre las diferentes opciones para recolectar datos con KoboToolbox, ver Recolección de datos con KoboToolbox.

Gestionar y descargar datos¶

Monitorea, revisa y exporta tus datos usando vistas de tabla flexibles y opciones de descarga.

  • Vistas de tabla: Personaliza las vistas, muestra los datos en varios idiomas y valida los envíos en tiempo real.

  • Descargar datos: Descarga y exporta datos en múltiples formatos, incluyendo XLS, CSV, KML, ZIP y GeoJSON.

Para obtener más información sobre la gestión de tus datos en KoboToolbox, ver Gestión de proyectos en KoboToolbox.

Visualizar y analizar datos¶

Identifica y comparte resultados con los informes integrados, los mapas interactivos y las herramientas de datos cualitativos de KoboToolbox.

  • Informes personalizados: Visualiza datos con informes personalizados y mapas interactivos.

  • Transcripción y traducción automatizadas: Usa funciones automatizadas para transcribir y traducir respuestas de audio y datos cualitativos.

  • Análisis cualitativo: Usa las funciones de análisis cualitativo asistido por IA para identificar temas y patrones en las respuestas de audio.

Para obtener más información sobre el análisis de tus datos en KoboToolbox, ver Análisis de datos con KoboToolbox.

Gestionar proyectos y equipos¶

Gestiona fácilmente una encuesta o cientos de ellas con las funcionalidades de gestión de proyectos de KoboToolbox, que incluyen monitoreo de proyectos, permisos para compartir y manejo de casos.

  • Resumen del proyecto: Monitorea los envíos entrantes, el acceso a la cuenta y la actividad del proyecto.

  • Permisos para compartir: Comparte proyectos con los miembros del equipo y controla quién puede enviar, ver y editar datos configurando diferentes permisos a nivel de usuario/a.

  • Manejo de casos: Vincula proyectos usando conexión dinámica de proyectos para recolectar y gestionar datos longitudinales.

  • Gestión de equipos: Centraliza la gestión de proyectos y usuarios para mejorar la supervisión y la colaboración en equipos grandes y distribuidos.

Para obtener más información sobre la gestión de tus proyectos en KoboToolbox, ver Gestión de proyectos en KoboToolbox.

Conectar KoboToolbox con otras plataformas¶

Conecta los datos del proyecto en diferentes plataformas e integra los datos de KoboToolbox con otros programas como Excel y Power BI.

  • API de KoboToolbox: Usa la interfaz de programación de aplicaciones (API) para crear informes interactivos y visualizar datos con otros programas.

  • Exportaciones sincrónicas: Comparte datos entre diferentes aplicaciones sin descargar ningún archivo usando exportaciones sincrónicas.

  • Servicios REST: Conecta tu proyecto a programas externos usando servicios REST.

Para obtener más información sobre cómo conectar KoboToolbox con otras plataformas, ver Integraciones.
