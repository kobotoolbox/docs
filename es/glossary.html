Glosario de KoboToolbox¶

Última actualización: 14 Jun 2026

Este glosario define los términos específicos de KoboToolbox relacionados con el diseño de formularios, la recolección de datos y la gestión de proyectos. Cada entrada incluye sinónimos relevantes, términos relacionados y enlaces a artículos de ayuda para obtener más información.

Cuenta¶

Perfil de usuario que permite acceder a KoboToolbox, incluyendo la capacidad de crear proyectos, gestionar datos y configurar ajustes personales.

Para más información, consulta Crear una cuenta en KoboToolbox.

Actividad de acceso a la cuenta¶

Los registros de acceso a la cuenta registran todos los inicios de sesión en tu cuenta de KoboToolbox. Ayudan a monitorear la seguridad de la cuenta al mostrar cuándo y desde dónde se accedió a ella.

Para más información, consulta Seguimiento a la actividad del proyecto y de la cuenta.

Término(s) relacionado(s): Cuenta, Registros del historial de la actividad del proyecto

Interfaz de programación de aplicaciones (API)¶

Conjunto de herramientas que permite que diferentes sistemas de software se comuniquen con KoboToolbox, habilitando acciones automatizadas como el envío de datos, la recuperación de datos o la gestión de proyectos de forma programática.

Para más información, consulta Introducción a la API.

Término(s) relacionado(s): Exportaciones sincrónicas

Apariencia¶

Configuración que cambia cómo se muestra una pregunta o grupo en un formulario, como mostrar varias preguntas en una sola pantalla o modificar la presentación de las opciones de respuesta.

Para más información, consulta Opciones de preguntas en el Formbuilder y Apariencia de preguntas en XLSForm.

Proyectos archivados¶

Proyectos que han sido retirados de la lista de proyectos activos y ya no aceptan nuevos envíos, pero se conservan como referencia o para el mantenimiento de registros.

Para más información, consulta Gestión de proyectos en KoboToolbox.

Término(s) relacionado(s): Proyectos implementados, Proyectos en borrador

Auditoría¶

Registro que hace seguimiento de acciones específicas realizadas durante la recolección de datos o la cumplimentación de un formulario, como el tiempo dedicado a cada pregunta o la ubicación, cuando los metadatos de auditoría están habilitados en el formulario.

Para más información, consulta Pregunta de metadatos de auditoría.

Término(s) relacionado(s): Metadatos

Autenticación¶

Proceso de verificación de la identidad de un usuario mediante la solicitud de un nombre de usuario y contraseña válidos para controlar el acceso a las cuentas, proyectos y envíos de KoboToolbox.

Grabación de sonido de fondo¶

Función que graba audio de forma continua durante una entrevista mientras se completa el formulario, capturando las respuestas habladas sin necesidad de preguntas de audio independientes.

Para más información, consulta Grabación de entrevistas con grabaciones de audio de fondo.

Ciclo de facturación¶

El período de tiempo antes de que tu plan de KoboToolbox se renueve y tu uso se reinicie. Para los usuarios del Community Plan, el ciclo de facturación es de un mes y comienza en la fecha en que se creó la cuenta. Para los planes de pago, el ciclo de facturación depende del tipo de suscripción: los planes mensuales se renuevan cada mes y los planes anuales se renuevan cada año. Puedes encontrar la fecha de creación de tu cuenta y, si corresponde, la fecha de renovación de tu plan en Configuración de la cuenta > Uso.

Para más información, consulta Configurar la cuenta.

Formularios en blanco (KoboCollect)¶

Formularios de recolección de datos vacíos que se han descargado en un dispositivo en la aplicación KoboCollect. Estos formularios aún no contienen respuestas y están listos para la recolección de datos. Los formularios en blanco también se denominan definiciones de formulario en la aplicación.

Para más información, consulta Recolección de datos con KoboCollect.

Término(s) relacionado(s): KoboCollect, Formularios guardados

Cálculo¶

Expresión automatizada en un formulario que genera un valor basado en otras respuestas o reglas predefinidas, sin que el encuestado tenga que ingresar el valor manualmente.

Para más información, consulta Agregar cálculos con el Formbuilder y Agregar cálculos a un XLSForm.

Término(s) relacionado(s): Funciones

Selección en cascada¶

Método para reducir las opciones de respuesta en varias preguntas de tipo Seleccionar una, donde las opciones de cada pregunta se filtran según la selección anterior del encuestado.

Para más información, consulta Agregar preguntas de selección en cascada en el Formbuilder y Agregar filtros de selección a un XLSForm.

Término(s) relacionado(s): Filtro de selección

Filtro de selección¶

Condición aplicada a una pregunta de tipo selección en XLSForm que limita la lista de opciones disponibles según una respuesta anterior u otra variable del formulario.

Para más información, consulta Agregar filtros de selección a un XLSForm.

Término(s) relacionado(s): Selección en cascada

Nombre de columna de datos¶

Nombres cortos internos asignados a las preguntas que se utilizan para almacenar y referenciar datos en un formulario. También se denominan nombres de pregunta en XLSForm, y a veces se les llama valores XML o encabezados en la interfaz de usuario. Exportar los datos usando valores y encabezados XML puede facilitar el manejo y análisis de datos.

Para más información, consulta Opciones de preguntas en el Formbuilder.

Término(s) relacionado(s): Etiqueta, Valor XML

Informes de datos¶

Informes automatizados en la vista DATOS > Informes que muestran los datos del proyecto en gráficos y tablas. Los informes se pueden personalizar para ajustar qué preguntas se muestran y cómo se visualizan los resultados.

Para más información, consulta Visualizar datos utilizando informes personalizados.

Idioma predeterminado¶

El idioma que se muestra por defecto cuando se abre un formulario, generalmente el idioma utilizado durante el diseño del formulario o el idioma principal previsto para la recolección de datos.

Para más información, consulta Recolectar datos en diferentes idiomas.

Valor predeterminado¶

Respuesta preestablecida que aparece automáticamente cuando se carga un formulario, permitiendo que los encuestados mantengan el valor tal como está o lo reemplacen con su propia respuesta.

Para más información, consulta Opciones de preguntas en el Formbuilder y Opciones de preguntas en XLSForm.

Implementar¶

Publicar un formulario para que quede activo y listo para la recolección de datos.

Para más información, consulta Implementar formularios para la recolección de datos.

Término(s) relacionado(s): Reimplementar

Proyectos implementados¶

Proyectos que han sido publicados y están listos para la recolección de datos, lo que permite a los usuarios abrir el formulario y enviar respuestas.

Para más información, consulta Gestión de proyectos en KoboToolbox.

Término(s) relacionado(s): Proyectos archivados, Proyectos en borrador

Borradores¶

Formularios que han sido iniciados pero aún no enviados al servidor. Los borradores se pueden reabrir, editar y finalizar más tarde. En KoboCollect, los borradores se almacenan en la sección Borradores de la aplicación. Al usar formularios web, los borradores se almacenan localmente en el navegador web.

Para más información, consulta Recolección de datos con KoboCollect y Recolección de datos a través de formularios web.

Término(s) relacionado(s): KoboCollect, Formularios guardados, Formularios web

Proyectos en borrador¶

Proyectos que aún están siendo desarrollados o editados y que aún no han sido implementados, por lo que no pueden recibir envíos.

Para más información, consulta Gestión de proyectos en KoboToolbox.

Término(s) relacionado(s): Proyectos archivados, Proyectos implementados

Conexión dinámica de proyectos¶

Función que permite a un formulario acceder a los envíos de otro proyecto de KoboToolbox, de modo que los campos de datos seleccionados se recuperan automáticamente y están disponibles durante la recolección de datos.

Para más información, consulta Conexión dinámica de proyectos en el Formbuilder y Conexión dinámica de proyectos usando XLSForms.

Cifrado¶

Función de seguridad que protege los envíos de formularios codificando los datos para que solo puedan ser descifrados por alguien que tenga la clave privada correcta.

Para más información, consulta Cifrar formularios.

Encuestador¶

Persona que recolecta datos completando un formulario durante una entrevista o visita de campo, ya sea en un dispositivo móvil o a través de un formulario web.

Término(s) relacionado(s): KoboCollect, Formularios web

Servidor con sede en la Unión Europea¶

El servidor de KoboToolbox alojado en la Unión Europea, que los usuarios pueden seleccionar para almacenar y gestionar sus proyectos y datos.

Para más información, consulta Crear una cuenta en KoboToolbox.

Término(s) relacionado(s): Servidor Global

Lógica del formulario¶

Conjunto de reglas y condiciones que controlan el comportamiento de un formulario, incluyendo la lógica de omisión, los criterios de validación, los cálculos y otras acciones automatizadas que responden a la entrada del usuario.

Para más información, consulta Introducción a la lógica de formularios en el Formbuilder y Introducción a la lógica de formularios en XLSForm.

Término(s) relacionado(s): Lógica de omisión, Criterios de validación, Filtro de selección, Cálculo

Formbuilder¶

Herramienta intuitiva dentro de la plataforma KoboToolbox que te permite crear, editar y organizar preguntas de formularios a través de una interfaz visual, sin necesidad de conocimientos técnicos.

Para más información, consulta Comenzar con el editor de formularios.

Término(s) relacionado(s): XLSForm

Funciones¶

Operaciones utilizadas en cálculos o expresiones de formularios para procesar datos, como realizar operaciones matemáticas, trabajar con fechas o modificar valores de texto.

Para más información, consulta Usar funciones en XLSForm.

Término(s) relacionado(s): Cálculo, Lógica del formulario

Reglamento General de Protección de Datos (RGPD)¶

Reglamento de protección de datos de la Unión Europea que establece las normas sobre cómo se deben recolectar, almacenar y procesar los datos personales, incluyendo los requisitos de privacidad, seguridad y consentimiento del usuario al manejar datos en KoboToolbox.

Para más información, consulta Reglamento General de Protección de Datos (RGPD).

Servidor Global¶

El servidor de KoboToolbox alojado en los Estados Unidos, que los usuarios pueden seleccionar para almacenar y gestionar sus proyectos y datos.

Para más información, consulta Crear una cuenta en KoboToolbox.

Término(s) relacionado(s): Servidor con sede en la Unión Europea

Sugerencia adicional¶

Instrucciones adicionales para una pregunta que están ocultas detrás de una sección expandible y que el encuestado puede abrir para ver orientación extra durante la recolección de datos.

Para más información, consulta Opciones de preguntas en el Formbuilder y Opciones de preguntas en XLSForm.

Término(s) relacionado(s): Sugerencia de pregunta

Etiquetas HXL (Lenguaje de intercambio del sector humanitario)¶

Hashtags estandarizados que se añaden a los conjuntos de datos para mejorar la interoperabilidad, permitiendo que los datos sean fácilmente comprendidos, combinados y analizados entre diferentes organizaciones humanitarias y herramientas.

Para más información, consulta Opciones de preguntas en el Formbuilder.

Kobo¶

La organización sin fines de lucro que desarrolla y mantiene KoboToolbox, apoyando la investigación global, la respuesta humanitaria y otros esfuerzos de bien social a través de herramientas de recolección de datos abiertas y accesibles.

Para más información, consulta Kobo.ngo.

Término(s) relacionado(s): KoboToolbox

KoboCollect¶

La aplicación Android KoboCollect utilizada para la recolección de datos en dispositivos móviles, que permite a los encuestadores descargar formularios, completarlos sin conexión y enviar los datos cuando tienen conexión. Es una de las dos formas de recolectar datos con KoboToolbox, junto con los formularios web.

Para más información, consulta Configuración de la aplicación KoboCollect.

Término(s) relacionado(s): Formularios web

KoboToolbox¶

Conjunto de herramientas para diseñar formularios, recolectar datos y gestionar proyectos, utilizado a nivel mundial para investigación, trabajo humanitario y otras iniciativas de bien social.

Para más información, consulta Acerca de KoboToolbox: recolección de datos accesible.

Término(s) relacionado(s): Kobo

Etiqueta¶

El texto que se muestra a los encuestados para una pregunta u opción de respuesta durante la recolección de datos. Las etiquetas se pueden traducir a varios idiomas.

Biblioteca¶

Colección de preguntas reutilizables o plantillas de formularios que se pueden agregar a nuevos proyectos, lo que te ayuda a diseñar formularios de manera más eficiente. Las bibliotecas también se pueden compartir con otros usuarios para uso colaborativo.

Para más información, consulta Usar la biblioteca de preguntas de KoboToolbox.

Término(s) relacionado(s): Colecciones públicas, Bloqueo de biblioteca

Bloqueo de biblioteca¶

Función que controla cómo se pueden editar las plantillas de la biblioteca después de utilizarlas para crear nuevos proyectos.

Para más información, consulta Bloquear cuestionarios para la biblioteca usando XLSForms.

Término(s) relacionado(s): Biblioteca

Metadatos¶

Los metadatos del formulario son información opcional que se añade a un envío y que proporciona contexto sobre cómo o cuándo se recolectaron los datos, como el ID del dispositivo, la hora de inicio y fin, o la ubicación. Los usuarios deben habilitar estos campos en el formulario para que sean recolectados.

Para más información, consulta Agregar metadatos en el Formbuilder y Metadatos en XLSForm.

Término(s) relacionado(s): Campos de envío generados por el sistema

Propietario¶

El usuario que tiene control total sobre un proyecto de KoboToolbox, incluyendo la gestión de sus datos y configuraciones, la edición del formulario, y la asignación o modificación de los permisos para compartir con otros usuarios.

Proyecto (KoboCollect)¶

En la aplicación KoboCollect, un proyecto hace referencia a una cuenta de KoboToolbox a la que está conectada la aplicación. Cada cuenta conectada aparece como su propio proyecto y puede contener varios formularios para la recolección de datos.

Para más información, consulta Configuración de la aplicación KoboCollect.

Término(s) relacionado(s): KoboCollect

Proyecto (KoboToolbox)¶

Un proyecto en KoboToolbox contiene un formulario, su configuración y todos los datos enviados a ese formulario, incluyendo cualquier archivo multimedia del proyecto.

Para más información, consulta Gestión de proyectos en KoboToolbox.

Término(s) relacionado(s): KoboToolbox

Permisos a nivel de proyecto¶

Configuraciones que controlan quién puede acceder a un proyecto y sus datos a nivel de proyecto, incluyendo opciones como permitir envíos sin autenticación, hacer público un proyecto, transferir la propiedad y eliminar proyectos compartidos.

Para más información, consulta Compartir proyectos con configuraciones a nivel de proyecto.

Término(s) relacionado(s): Permisos a nivel de usuario

Registros del historial de la actividad del proyecto¶

Los registros del historial de la actividad del proyecto proporcionan un registro detallado de la actividad del proyecto, incluyendo actualizaciones del formulario, ediciones de datos y cambios en la configuración del proyecto.

Para más información, consulta Seguimiento a la actividad del proyecto y de la cuenta.

Término(s) relacionado(s): Proyecto (KoboToolbox), Actividad de acceso a la cuenta

Página de proyectos¶

La página de proyectos es el panel principal que ves por defecto cuando inicias sesión en tu cuenta de KoboToolbox. Muestra todos tus proyectos y detalles clave como el estado, la propiedad, las fechas de última modificación y el número de envíos, con acceso rápido a las acciones del proyecto.

Para más información, consulta Gestión de proyectos en KoboToolbox.

Colecciones públicas¶

Función que permite a los usuarios compartir públicamente plantillas de formularios o preguntas seleccionadas de su biblioteca, poniéndolas a disposición de cualquier persona para que las explore y reutilice.

Para más información, consulta Colecciones públicas en la biblioteca KoboToolbox.

Término(s) relacionado(s): Biblioteca

Extracción de datos¶

Función que permite a un formulario recuperar valores de un archivo CSV externo durante la recolección de datos, llenando automáticamente campos específicos basándose en un valor coincidente.

Para más información, consulta Extraer datos de un archivo CSV externo.

Grupo de preguntas¶

Conjunto de preguntas relacionadas agrupadas en un formulario para mejorar su estructura y organización. Los grupos de preguntas se pueden mostrar en la misma pantalla, omitir como unidad usando lógica de omisión, o repetir para recolectar múltiples conjuntos de respuestas.

Para más información, consulta Grupos y grupos de repetición en el Formbuilder y Grupos de preguntas en XLSForm.

Término(s) relacionado(s): Grupo de repetición

Referencia a preguntas¶

Uso de las respuestas de una pregunta en otras partes del formulario, como mostrar una respuesta anterior en el texto de preguntas posteriores o utilizarla en cálculos, lógica de omisión o criterios de validación.

Para más información, consulta Introducción a la lógica de formularios en el Formbuilder y Introducción a la lógica de formularios en XLSForm.

Término(s) relacionado(s): Lógica del formulario

Tipo de pregunta¶

El tipo de pregunta utilizado en un formulario, que determina cómo los encuestados proporcionan sus respuestas, como texto, número, seleccionar una o calificación.

Para más información, consulta Agregar preguntas con el Formbuilder y Tipos de preguntas en XLSForm.

Reimplementar¶

Actualizar un formulario ya implementado después de realizar cambios, reemplazando la versión implementada anterior para que el formulario actualizado pueda utilizarse en la recolección de datos en curso.

Para más información, consulta Implementar formularios para la recolección de datos.

Término(s) relacionado(s): Implementar

Sugerencia de pregunta¶

Información adicional que se muestra directamente debajo de la etiqueta de una pregunta para ayudar a los encuestados o encuestadores a entender cómo responder. Las sugerencias de pregunta siempre son visibles en el formulario. También se denominan sugerencias comunes en el Formbuilder.

Para más información, consulta Opciones de preguntas en el Formbuilder y Opciones de preguntas en XLSForm.

Término(s) relacionado(s): Sugerencia adicional

Grupo de repetición¶

Conjunto de preguntas que se pueden responder varias veces dentro de un formulario, lo que permite a los encuestados ingresar información repetida como miembros del hogar, artículos o eventos.

Para más información, consulta Grupos y grupos de repetición en el Formbuilder y Grupos repetidos en XLSForm.

Término(s) relacionado(s): Grupo de preguntas

Servicios REST¶

Función que permite a KoboToolbox enviar datos automáticamente a sistemas externos, enviando nuevos envíos a un endpoint REST especificado cada vez que se reciben.

Para más información, consulta Usar servicios REST.

Formularios guardados (KoboCollect)¶

Formularios que contienen datos ingresados desde la aplicación KoboCollect. Esto incluye borradores, formularios finalizados y formularios que ya han sido enviados al servidor.

Para más información, consulta Recolección de datos con KoboCollect.

Término(s) relacionado(s): KoboCollect, Formularios en blanco, Borradores

Lógica de omisión¶

Reglas que controlan si una pregunta o grupo aparece en el formulario, según las respuestas anteriores u otras condiciones. La lógica de omisión también se denomina condiciones de relevancia.

Para más información, consulta Añadir lógica de salto al Formbuilder y Añadir lógica de salto a un XLSForm.

Término(s) relacionado(s): Lógica del formulario

Envío¶

Entrada de formulario completada y enviada a un proyecto, que contiene todas las respuestas y cualquier archivo multimedia asociado. Un envío también se denomina registro.

Exportaciones sincrónicas¶

Método de exportación de datos que se actualiza automáticamente a medida que llegan nuevos envíos, lo que permite que herramientas externas como Power BI, Excel o Google Sheets accedan siempre a los datos más recientes sin necesidad de descargas manuales.

Para más información, consulta Usar la API para exportaciones sincrónicas.

Término(s) relacionado(s): API

Campos de envío generados por el sistema¶

Campos que KoboToolbox crea automáticamente para cada envío, como _id, _uuid, _index, _submission_time, _submitted_by y rootUuid. Estos campos ayudan a identificar registros y hacer seguimiento de los detalles del envío, aparecen automáticamente en la tabla de datos y en las exportaciones, y no pueden ser editados por los usuarios.

Para más información, consulta Ver y validar datos.

Término(s) relacionado(s): Metadatos

Permisos a nivel de usuario¶

Configuraciones que controlan lo que usuarios específicos pueden hacer dentro de un proyecto, como ver el formulario, agregar envíos, editar datos, validar envíos o gestionar la configuración del proyecto.

Para más información, consulta Compartir proyectos con permisos a nivel de usuario/a.

Término(s) relacionado(s): Permisos a nivel de proyecto

Criterios de validación¶

Reglas que definen qué se considera una respuesta aceptable para una pregunta, como requerir que un número esté dentro de un rango determinado. Los criterios de validación también se denominan restricciones.

Para más información, consulta Añadir criterios de validación en el Formbuilder y Agregar restricciones a un XLSForm.

Término(s) relacionado(s): Lógica del formulario

Formularios web¶

Formularios basados en navegador utilizados para recolectar datos en línea o sin conexión, que ofrecen una alternativa a la aplicación Android KoboCollect y son compatibles con todos los dispositivos principales.

Para más información, consulta Recolección de datos a través de formularios web.

Término(s) relacionado(s): KoboCollect

XLSForm¶

Estándar de diseño de formularios que utiliza hojas de cálculo de Excel para definir preguntas, opciones y lógica del formulario, proporcionando una forma estructurada de crear formularios complejos o avanzados para KoboToolbox.

Para más información, consulta Introducción a XLSForm.

Término(s) relacionado(s): Formbuilder

Formulario XML¶

La versión XML de un formulario que se genera cuando se implementa un proyecto, utilizada por las herramientas de recolección de datos para ejecutar el formulario y aplicar su estructura, lógica y configuración. Un formulario XML también se denomina XForm.

Término(s) relacionado(s): XLSForm

Valor XML¶

Nombres cortos internos asignados a las opciones que se utilizan para almacenar y referenciar datos en un formulario. También se denominan nombres de opción en XLSForm. En la interfaz de usuario, el término valor XML a veces se utiliza para referirse a los nombres de columna de datos. Exportar los datos usando valores XML puede facilitar el manejo y análisis de datos.

Para más información, consulta Configurar valores XML para las opciones de respuesta.

Término(s) relacionado(s): Etiqueta, Nombre de columna de datos
